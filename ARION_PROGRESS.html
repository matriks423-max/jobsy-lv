Manual Steps — Do These When You Get Home
Do these in order. Steps 1–3 must be done before anything runs. Steps 4–9 are account creation — open them in separate tabs simultaneously to save time.
○
1. Move code to arion-word repo manual
Create a classic PAT at github.com/settings/tokens (tick repo) → add as secret ARION_WORLD_PAT in jobsy-lv → run "Migrate — Copy Arion World" workflow in Actions tab → delete the secret after.
○
2. Fix Formspree placeholder manual
Create free form at formspree.io → open website/src/components/ContactForm.astro → replace FORMSPREE_FORM_ID with your actual ID.
○
3. Enable GitHub Pages on arion-word repo manual
arion-word → Settings → Pages → Source → GitHub Actions → Save.
○
4. Create API accounts manual
Anthropic (API key) · ElevenLabs (API key + Voice ID) · Replicate (API token) · DeepL (free key) · Printful (API key) · Formspree (Form ID)
○
5. Create social channels manual
YouTube channel @ArionWorld · Facebook Page "Arion World" · Instagram Business @ArionWorld (link to FB Page) · TikTok creator @ArionWorld
○
6. Get YouTube OAuth token manual
Google Cloud Console → Enable YouTube Data API v3 → OAuth 2.0 credentials (Desktop) → run python src/youtube_auth.py locally → copy printed YOUTUBE_REFRESH_TOKEN.
○
7. Get Facebook/Instagram tokens manual
developers.facebook.com → Create App (Business) → add Instagram Graph API + Pages API → generate long-lived Page Access Token → copy: FACEBOOK_PAGE_ID, FACEBOOK_ACCESS_TOKEN, FACEBOOK_APP_ID, FACEBOOK_APP_SECRET, INSTAGRAM_ACCOUNT_ID.
○
8. Get TikTok tokens manual
developers.tiktok.com → Create app → enable Content Posting API → OAuth flow → copy: TIKTOK_ACCESS_TOKEN, TIKTOK_REFRESH_TOKEN, TIKTOK_CLIENT_KEY, TIKTOK_CLIENT_SECRET.
○
9. Create Discord server + bot manual
discord.com/developers → New Application → Bot → copy DISCORD_BOT_TOKEN → Create server → #announcements channel → copy DISCORD_ANNOUNCEMENT_CHANNEL_ID → invite bot with Send Messages + Embed Links permissions.
○
10. Add all secrets to arion-word repo manual
arion-word → Settings → Secrets → Actions → add all 22 secrets (see MANUAL_ACTIONS_REQUIRED.md for full list).
○
11. Test dry run manual
arion-word → Actions → "Arion World — Weekly Episode" → Run workflow → dry_run: true. Should print episode title, scene count, cliffhanger.
