OAuth Token Flows (one-time)
✓
Get YouTube refresh token One-time OAuth
Run this once locally — gives the pipeline a long-lived token to upload videos.
▸ Steps
1
Clone the repo locally and run:
cd story-publisher && pip install google-auth-oauthlib
2
Run python src/youtube_auth.py — it opens a browser, sign in with the YouTube account
3
After authorising, the script prints a refresh_token — copy it → YOUTUBE_REFRESH_TOKEN
This token doesn't expire unless you revoke it. The pipeline uses it to get fresh access tokens automatically.
✓
Get Facebook Page long-lived access token One-time OAuth
Facebook page tokens last 60 days — the monthly-token-refresh workflow renews them automatically.
▸ Steps
1
Go to Meta Graph API Explorer
2
Select your app → Generate User Access Token → Permissions: pages_manage_posts, pages_read_engagement, instagram_basic, instagram_content_publish
3
Exchange for long-lived token: GET /oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT_TOKEN
4
Then get Page token: GET /me/accounts — find your page → copy access_token → FACEBOOK_ACCESS_TOKEN
5
Get INSTAGRAM_ACCOUNT_ID: GET /me?fields=instagram_business_account
✓
Get TikTok access + refresh token One-time OAuth
TikTok tokens refresh monthly — the monthly-token-refresh workflow handles renewal.
▸ Steps
1
In TikTok Developer Portal, go to your app → Auth → generate the OAuth URL with your client_key and scope video.upload
2
Open the URL in a browser, log in as your TikTok account, authorise
3
Copy the code from the redirect URL, exchange it: POST /v2/oauth/token/
4
Response gives access_token → TIKTOK_ACCESS_TOKEN and refresh_token → TIKTOK_REFRESH_TOKEN
